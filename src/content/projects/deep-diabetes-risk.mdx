## Overview

Deep Diabetes Risk is a machine learning project that predicts binary diabetes status from 21 tabular health indicators sourced from the CDC BRFSS 2015 survey dataset. The project compares baseline models against a custom Deep MLP built in PyTorch, with full preprocessing, training, evaluation, and visualization pipelines.

## Role

ML Engineer — built the Deep MLP architecture and training pipeline, and handled model evaluation including metrics comparison, ROC/PR curves, confusion matrix, feature ablation, and publication-ready plots against baseline models.

## Problem

Diabetes affects millions of people, and early risk prediction from routine health survey data could help flag at-risk individuals before clinical diagnosis. The challenge was building a model that handles real-world class imbalance and generalizes well across diverse health indicators.

## Solution

Trained a Deep MLP using PyTorch with weighted BCE loss to handle class imbalance, AdamW optimizer, and a CosineAnnealingLR scheduler. Used early stopping based on validation ROC-AUC to prevent overfitting. Evaluated the model against Logistic Regression, KNN, and Gaussian Naive Bayes baselines across multiple metrics.

## Technical Details
- Dataset: CDC BRFSS 2015 — 21 health indicator features, binary diabetes classification
- Deep MLP built in PyTorch with weighted BCE loss for class imbalance handling
- Optimizer: AdamW with CosineAnnealingLR scheduling and early stopping on validation ROC-AUC
- Stratified 70/15/15 train/val/test split with standardized numeric features
- Evaluation outputs: ROC curves, PR curves, calibration plots, confusion matrix, feature ablation, and metrics comparison table across all models
- Baseline comparisons: Logistic Regression, KNN, Gaussian Naive Bayes (via Scikit-learn)
